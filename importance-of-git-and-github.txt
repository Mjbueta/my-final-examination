In this generation where technologies and many platforms are develop and one of the popular or common used in web-based platform- version control is git together with GitHub. Git is the tool to manage the source of the project while GitHub is the body in order to help to manage git.  The importance of this is that it helps to be more precise in conducting a project to less effort and time and enable to be easier to the developer to use. Git can be use without GitHub but GitHub is necessary because it is a host where the project comes in a variety forms of different languages. So git and GitHub its original purpose was to help groups of developers work collaboratively on big software projects in order to solve the problem easily.
